'weechat: automatic identify to NickServ and BitlBee' tags: - irc - linux status: publish type: post published: true meta: {} --- If you don't want to type on each start of weechat to identify yourself, do: /set freenode.server_command="/msg NickServ identify PASSWORD" Replace freenode with the server you want to connect of course if needed. For automaticaly connect to the BitlBee-Server you have to type: /set bitlbee.server_command="/msg &bitlbee identify PASSWORD"
