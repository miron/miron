'xmms2: ERROR: Could not connect to xmms2d: xmms2d is not running.' tags: - audio - linux status: publish type: post published: true meta: {} --- I came around this error, it means that the daemon and the client don't run in the same directory. xmms2d -i unix:///tmp/apantutarinte export XMMS_PATH=unix:///tmp/apantutarinte xmms2 list helps alot. Edit: this is no more necessary with DrGonzo release.
