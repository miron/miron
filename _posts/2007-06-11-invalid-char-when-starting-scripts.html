--- layout: post title: Invalid char when starting scripts tags: - ruby - unicode - vim status: publish type: post published: true meta: {} --- foobar.rb:1: Invalid char `\357' in expression foobar.rb:1: Invalid char `\273' in expression foobar.rb:1: Invalid char `\277' in expression Your script gives you this error and and vim shows nothing ? Try to set :set nobomb and save the file. When you start vim in binarymode, vim -b foobar.rb, than you see the invalid chars
